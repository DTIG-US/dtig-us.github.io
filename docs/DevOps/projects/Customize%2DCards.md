   2. Add Start Date
   3. Add Target Date
     a. Select Check if you want to display these fields


 - Styling Rules
   1.  Rule Name: Late Start 
     a. Card color: Yellow 
     b. Start Date < @Today 
     c. State = Proposed

   2. Rule Name: On Time
    a. Card color: Orange
    b. Start Date < @Today
    c. State <> Proposed

   3.  Rule Name: Late
     a. Card color: Red
     b. Title Style: Bold
     c. Target Date <= @Today
     d. State = Active

   4. Rule Name: Complete
     a. Card color: Green
     b. State = Closed
